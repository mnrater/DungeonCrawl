# DungeonCrawl
Roguelike game made in Java and JavaFX.

Main functionalities:
- picking up items <br>
- fighting with many enemy types <br>
- getting a key and opening doors <br>
- movable enemies <br>
- saving game state <br>

Technologies:
- Backend: Java, JavaFX; <br>
- Database: PostgreSQL. <br>
